기본설정
패널의 기본 환경설정 페이지입니다.
팬카페 패널 타이틀
팬카페 패널 상단에 들어가는 제목입니다. 20자 이내를 권장합니다. 적용된 테마에 따라 변경한 제목이 반영되지 않을 수 있습니다.
네이버카페 ID
게시판을 가져올 네이버카페의 고유ID입니다(네이버 로그인 ID 아님.) 게시판을 가져올 네이버카페 주소창에서 "search.clubid"의 숫자로 구성된 ID를 복사해 붙여넣어주세요.
트게더 ID
게시판을 가져올 트게더의 고유ID입니다. 게시판을 가져올 트게더의 "https://tgd.kr/s/{고유ID}" 주소에서 {고유ID} 부분을 복사해 붙여넣어주세요.
로딩페이지 로딩문구
게시판 로딩페이지에서 띄울 로딩문구입니다.
로딩 실패시 문구
게시판 로딩 실패시 띄울 문구입니다.
링크복사 알림 문구
게시글을 클릭하여 링크를 복사했을 때 띄울 문구입니다.
