기본설정
패널의 기본 환경설정 페이지입니다.
팬카페 패널 타이틀
팬카페 패널 상단에 들어가는 제목입니다. 20자 이내를 권장합니다. 적용된 테마에 따라 변경한 제목이 반영되지 않을 수 있습니다.
네이버카페 ID
게시판을 가져올 네이버카페의 고유ID입니다(네이버 로그인 ID 아님.) 게시판을 가져올 네이버카페 주소창에서 "search.clubid"의 숫자로 구성된 ID를 복사해 붙여넣어주세요.
트게더 ID
게시판을 가져올 트게더의 고유ID입니다. 게시판을 가져올 트게더의 "https://tgd.kr/s/{고유ID}" 주소에서 {고유ID} 부분을 복사해 붙여넣어주세요.
로딩페이지 로딩문구
게시판 로딩페이지에서 띄울 로딩문구입니다.
로딩 실패시 문구
게시판 로딩 실패시 띄울 문구입니다.
링크복사 알림 문구
게시글을 클릭하여 링크를 복사했을 때 띄울 문구입니다.
API URL
게시판 데이터를 가져올 API의 URL입니다.
기본설정 초기화

설정 초기화는 저장 버튼을 통해 저장 후에 적용됩니다. 저장 이전에 새로고침을 하면 초기화 이전 값으로 돌아가게 됩니다.

게시판설정
게시판 환경설정 페이지입니다.
최대 4개의 "네이버카페", "트게더" 게시판을 등록할 수 있습니다.
게시판1
게시판명
탭에 표시될 게시판의 이름입니다.
플랫폼
게시판을 가져올 플랫폼입니다. [기본설정]에서 사용할 플랫폼의 ID 설정이 되어있어야 합니다.
게시판ID
가져올 게시판의 ID입니다. 네이버카페는 게시판 페이지 주소 내 search.menuid의 값을 복사하여 붙여넣습니다. 트게더는 주소 내 category의 값을 복사하여 붙여넣습니다.
게시글개수
가져올 게시판의 개시글 최대 개수입니다(공지글은 이 개수에 포함되지 않습니다.)
공지글숨기기
공지글 숨기기를 활성화하면 게시글을 가져올 때 공지글은 가져오지 않습니다.
